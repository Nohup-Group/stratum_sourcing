# Artifact Digests

This directory holds curated, reviewable summaries of artifacts and attachments.

## Rules

- Use these digests before opening raw files.
- Keep provenance in every digest.
- Do not treat the digest as stronger evidence than the underlying artifact.

Generated files such as `workspace-artifacts.md` and `email-attachments.md` are
refreshed during Phase 1 bootstrap from the live workspace and transcript corpus.
