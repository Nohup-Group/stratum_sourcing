# Stratum Artifacts

Use this skill when the task is about decks, attachments, PDFs, DOCX files,
workspace files, or historical artifacts.

## Read first

- `knowledge/artifacts/workspace-artifacts.md`
- `knowledge/artifacts/email-attachments.md`
- any other relevant file in `knowledge/artifacts/`

## Workflow

1. Start with the curated digest.
2. Only open the raw artifact if the digest is insufficient.
3. Keep provenance attached to any extracted fact.

## Rules

- Do not describe artifact contents you have not actually inspected or cited.
- If the digest suggests a likely mapping between an attachment and a local file, label it as an inference unless explicitly verified.
